### Prerequisites

- [Hammerspoon](https://www.hammerspoon.org/)
- [Bun](https://bun.sh/)

### Workflow

1. **Clone and Setup**:

   ```bash
   git clone https://github.com/MuXiu1997/dot-hammerspoon.git .
   bun install
   ```

2. **Build**:
   - `bun run build`: One-time compilation.
   - `bun run dev`: Watch mode for active development.

### Automated Deployment

The compiled Lua scripts are automatically deployed to the `dist` branch via GitHub Actions. This allows for a clean separation between the TypeScript source and the final Lua configuration used by Hammerspoon.
